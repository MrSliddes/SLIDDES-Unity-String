https://docs.unity3d.com/Manual/upm-ui-giturl.html

## How To Use It
1. Add ```using SLIDDES.String``` at the top of your script.
2. You can use the functions by ```"fooBar".FunctionName()``` or ```STR.FunctionName()```
3. Thats it!
